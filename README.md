# Data Analysis and Engineering

This project analyses and visualises data from various open datasets using ETL script in Python. 

Tools used: 
1. SQL
2. Python (Pandas)
3. Tableau
4. Google Sheets

## What does it do?
1. Loads the data from local csv files and puts them into PostgreSQL in multiple tables. 
2. Analysis of the datasets using multiple SQL queries.
3. Converting the results of the analysis to Pandas dataframe and exporting them into Google Sheets.
4. Visualisation in Tableau using Google Sheets connection - Scatter plot, Pie charts, Bar charts, etc.
